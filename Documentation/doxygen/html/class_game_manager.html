Detailed Description

Class which handles the main loop, timing, and game logic.

Calls drawing and updating functions explicitly on the Player and non-enemy Entities. Enemies are fully handled and owned by WaveManager; GameManager calls WaveManager's drawing and updating functions to control Enemies.
