\doxysubsection{Detailed Description}
Class which handles the main loop, timing, and game logic.

Calls drawing and updating functions explicitly on the \mbox{\hyperlink{class_player}{Player}} and non-\/enemy Entities. Enemies are fully handled and owned by \mbox{\hyperlink{class_wave_manager}{Wave\+Manager}}; \mbox{\hyperlink{class_game_manager}{Game\+Manager}} calls \mbox{\hyperlink{class_wave_manager}{Wave\+Manager}}\textquotesingle{}s drawing and updating functions to control Enemies.
