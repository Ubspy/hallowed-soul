\doxysubsection{Detailed Description}
Class which is used by \mbox{\hyperlink{class_game_manager}{Game\+Manager}} to spawn and update hoards of enemies.

\mbox{\hyperlink{class_wave_manager}{Wave\+Manager}} exclusively owns the enemies of the game, so it is calling the drawing and updating functions for them.
